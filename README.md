## Como rodar a aplicação :arrow_forward:

No terminal, clone o projeto: 

```
git clone https://github.com/Gileno29/gestor-tarefas.git
```
Baixe as dependencias:

```
go mod tidy
```
Rode a aplicação:

```
go run main.go
```
